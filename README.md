# Asteroid Kaboom Game

## Overview
Interactive Python game built using turtle graphics where players destroy randomly generated asteroids by guessing their coordinates.

## Features
- Randomly generated star field and asteroid positions
- Two asteroid types with different point values
- Coordinate-based user input system
- Visual explosion effects and scoring feedback

## How It Works
- The game generates a space field with stars and asteroids
- The player inputs coordinates to "shoot"
- Points are awarded based on hitting asteroids:
  - Blue asteroid: 100 points
  - Pink asteroid: 200 points
- Final score is displayed at the end

## Technologies
- Python
- Turtle Graphics
- Random module

## How to Run
1. Make sure Python is installed
2. Run the script: asteroid_kaboom.py
